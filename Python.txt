1) Создать переменную типа String (Строка)
 - представляет собой последовательность символов, для создания используются одинарные или двойные кавычки
a = "Welcome"

2) Создать переменную типа Integer (Числа)
 - целые числа
b = 2

3) Создать переменную типа Float (Числа с плавающей запятой)
 - числа с плавающей запятой имеют ограниченную точность
c = 3.5

4) Создать переменную типа Bytes (Байтовые строки)
 - последовательность чисел в диапазоне 0-255
d = b'232'

5) Создать переменную типа List (Список)
 -  представляет собой упорядоченную последовательность элементов
e = [5,"list",5**2]

6) Создать переменную типа Tuple (Кортеж)
 - так же как и список (list), является упорядоченной последовательностью элементов. Вся разница заключается в том, что кортежи неизменяемы
f = (5,"tuple",5**2)

7) Создать переменную типа Set (Множества)
 - неупорядоченнная коллекция уникальных объектов
g = {15, 25, 35, 45, 15}

8) Создать переменную типа Frozen set (Неизменяемые множества)
 - то же самое, что и set, только не может изменяться (immutable)
h = frozenset({15, 25, 35, 45, 15})

9) Создать переменную типа Dict (Словарь)
 - неупорядоченные наборы пар ключ-значение, используются, когда нужно сопоставить каждому из ключей значение и иметь возможность быстро получать доступ к значению, зная ключ
i = {"city": "Moscow"}

10) Вывести в консоль все выше перечисленные переменные с добавлением типа данных.
print(a,type(a)) - Welcome <class 'str'>
print(b,type(b)) - 2 <class 'int'>
print(c,type(c)) - 3.5 <class 'float'>
print(d,type(d)) - b'232' <class 'bytes'>
print(e,type(e)) - [5, 'list', 25] <class 'list'>
print(f,type(f)) - (5, 'tuple', 25) <class 'tuple'>
print(g,type(g)) - {25, 35, 45, 15} <class 'set'>
print(h,type(h)) - frozenset({25, 35, 45, 15}) <class 'frozenset'>
print(i,type(i)) - {'city': 'Moscow'} <class 'dict'>
